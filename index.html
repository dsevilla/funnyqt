FunnyQT

Functional Model Queries and Transformations

Download this project as a .zip file Download this project as a tar.gz file

Welcome!

FunnyQT is a model querying and transformation library designed as an API and embedded DSL in Clojure. It supports JGraLab graphs and EMF models.

For getting further information about FunnyQT visit its project page.
